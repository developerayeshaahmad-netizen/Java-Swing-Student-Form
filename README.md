Java-Swing-Student-Form
A Java Swing-based Student Registration Form that collects user input such as name, roll number, class, and email, with basic validation and interactive GUI components.

A simple and user-friendly Student Registration Form built using Java Swing. This application allows users to enter student details and performs basic input validation before submission.

Features:
Graphical User Interface (GUI) using Java Swing
Input fields for:
Name
Roll Number
Class
Email

Input validation:

-Name must start with a capital letter
-Roll number must contain only digits
-Class must be in uppercase letters
-Email must follow a basic valid format
-Submit button with success message
-Reset button to clear all fields

Built With:
-Java
-Java Swing (GUI)
-AWT (for event handling and styling)
